```python
def _filter_low_amplitude_peaks(
    peak_ts: np.ndarray,          # sorted peak timestamps in epoch-ms
    ecg_signal: np.ndarray,       # full ECG signal
    timestamps_ms: np.ndarray,    # sample timestamps aligned with ecg_signal
) -> tuple[np.ndarray, int]:
    """Remove false-positive peaks whose amplitude is much lower than neighbors.
    
    Returns: (filtered_peak_ts, n_removed)
    """
```

**Logic**:
1. Look up raw ECG amplitude at each peak via binary search on timestamp indices
2. Compute global median amplitude and 20% floor threshold
3. For each peak:
   - Reject if below global floor
   - Search for neighboring peaks within ±400ms
   - Reject if peak amplitude < 30% of max neighbor amplitude
4. Return filtered peak array and count of removed peaks
